# Money Minder

A lightweight personal finance app to track everyday spending, transactions, and receipt photos.

## Features

- Add transactions with date, amount, category, merchant, and notes.
- Upload a receipt image per transaction.
- Persist all data in browser `localStorage`.
- Search transactions by merchant or notes.
- Filter transactions by category.
- View spending summary for all time and current month.
- Delete transactions.

## Run locally

No build tools are required.

1. Open `index.html` in your browser.
2. Start adding transactions.

> Note: Receipt photos are saved in local browser storage as base64 image data.
